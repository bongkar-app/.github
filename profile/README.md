## What is bongkar?

**bongkar** *(Indonesian: to dismantle, to unpack)* is an automated APK reverse engineering platform. Upload an Android APK and get back a detailed analysis — decompiled source code, extracted API endpoints, hardcoded secrets, and native binary inspection.

Built for security researchers, developers, and penetration testers who need fast, reliable APK analysis without the manual overhead.
